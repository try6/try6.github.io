# Local Chinese Post Drafts

This directory stores local Chinese versions of posts.

Files here are intentionally outside `content/`, so Hugo will not publish them,
include them in post lists, RSS feeds, or the sitemap.

Recommended convention:

- Published English post: `content/posts/<slug>/index.md`
- Local Chinese version: `local/zh/posts/<slug>.md`

Keep the slug identical between the two files so each pair is easy to find.
